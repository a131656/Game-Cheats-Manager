Game Cheats Manager 为玩家提供了一个高效管理单机游戏修改器的一站式解决方案。它允许用户在一个便捷的位置浏览、下载和管理所有修改器。每个修改器，通常是一个独立的可执行文件，都可以通过应用程序直接启动或删除，从而简化您的游戏体验。

## 使用

1. **浏览修改器**: 在左侧使用搜索栏或浏览列表查找已下载的修改器。双击或点击 `启动` 按钮以启动修改器；点击 `删除` 以删除修改器。
2. **下载修改器**: 在右侧栏中，使用关键词搜索并按下 `回车` 键以浏览可用修改器，双击所需的匹配项以直接下载。修改器下载路径显示于右侧栏底部，您可以点击右侧的 `...` 来更改它。
3. **修改器管理**: `修改器管理` 面板为您提供了每个修改器来源的所有设置。您可以找到诸如自动更新修改器及修改器检索数据或更改下载服务器等设置。
4. **选项**: `选项` 菜单栏包含以下功能：
   1. **设置**: 调整诸如主题和语言等设置。
   2. **导入修改器**: 从文件选择窗口选择您想要导入的修改器。导入的修改器不支持自动更新。
   3. **打开修改器下载路径**: 查看修改器下载文件夹。
   4. **添加路径至白名单**: 将修改器下载路径添加到 Windows Defender 排除项中。如果您安装了其他杀毒软件，则可以手动执行此操作。
   5. **关于**: 查看应用版本和项目相关链接。

## 安装

1. **下载安装程序**: 前往 [latest release](https://github.com/dyang886/Game-Cheats-Manager/releases) 并下载适用于 Windows 的安装程序 (64-bit)。
2. **运行安装程序**: 运行安装程序，并按照说明来安装 Game Cheats Manager。
3. **启动应用程序**: 从应用程序文件夹或开始菜单中打开 Game Cheats Manager。
